## Will You Understand Code?

- **ഇംഗ്ലീഷ്:** After finishing all 9 lessons, you will **understand** most Python scripts used in DevOps (reading configs, calling APIs, running commands, handling errors).
- **മലയാളം:** 9 ലെസ്സനുകളും ചെയ്താൽ ഡെവോപ്സിൽ ഉപയോഗിക്കുന്ന പൈത്തൺ കോഡ് **മനസ്സിലാക്കാൻ** പോരാം.

## Can You Pass a Python Session in DevOps?

- **ഇംഗ്ലീഷ്:** For a **beginner / “Python for DevOps”** session, this material is **usually enough** to pass if you:
  1. Read and **practice every example** (type and run).
  2. Write 3–5 small scripts yourself (e.g. read a JSON file, run a command, call an API).
  3. Revise **01 to 04** (basics, loops, functions, list/dict) – these are asked most.
- **മലയാളം:** **ബിഗിനർ / DevOps-നുള്ള പൈത്തൺ** സെഷനാണെങ്കിൽ, മെറ്റീരിയൽ **പൊതുവേ പോരും**. പാസ് ചെയ്യാൻ: എല്ലാ ഉദാഹരണവും പ്രാക്ടീസ് ചെയ്യുക, 3–5 സ്ക്രിപ്റ്റ് സ്വയം എഴുതുക, 01–04 നല്ലപോലെ റിവൈസ് ചെയ്യുക.

## If Your Session Has Extra Topics

- **Testing (pytest):** They might ask you to write a small test. Learn: `def test_something(): assert add(1,2) == 3`.
- **More regex:** Session might use `re.search`, `re.findall` for logs – revise the `re` part in **09_devops_scripts.md**.
- **YAML:** If they use YAML configs, you already have a short note in **09**; install `pyyaml` and try once.
